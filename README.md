# Quote Fullstack App

A simple fullstack application built with Flutter and Node.js.

## 🚀 Features
- Get random quotes
- Add new quotes
- Delete quotes

## 🧱 Tech Stack
### Frontend
- Flutter
- Riverpod
- Dio

### Backend
- Node.js
- Express

## ⚙️ How to Run

### Backend
cd backend_node
npm install
node index.js

### Frontend
cd frontend_flutter
flutter pub get
flutter run

## 🌐 API Endpoints
- GET /quotes
- GET /quotes/random
- POST /quotes
- DELETE /quotes/:id
